Guided chirpy project from Boot.dev

This is a super simple version of twitter.  It runs a server that users can send get and post requests to create and recieve "chrips."

Requirements:

1.  Go  
2.  Postgres v15 or later  
3.  Install Goose: go install github.com/pressly/goose/v3/cmd/goose@latest  
    From sql/schema directory run: goose postgres "postgres://postgres:postgres@localhost:5432/gator" up  
4.  Install SQLC: go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest  
    Run: sqlc generate  
    This will generate our sql models  

Setup:

  Create a .env file in the main repo directory:

  Secret should be generated by running: `openssl rand -base64 64`

  Polka Key can and should be changed to another value

```
DB_URL="postgres://postgres:postgres@localhost:5432/chirpy?sslmode=disable"
PLATFORM="dev"
SECRET=""
POLKA_KEY="f271c81ff7084ee5b99a5091b42d486e"
```

Install:
Run: `go install` Bootdev_Chirpy
or
Run: `go build -o chirpy` and use chirpy
